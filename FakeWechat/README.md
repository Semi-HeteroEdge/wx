# FakeWechat
高仿微信界面，主要包含联系人界面，聊天界面，主要组件：跟随手指自适应弹出菜单

## 包含以下主要功能
####  跟随手指位置自适应弹出菜单
#### 标记未读
####  置顶聊天
#### 自动回复消息
#### 自动添加Time Stamp
